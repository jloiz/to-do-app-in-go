# to-do-app-in-go
A learning repo to create a simple CRUD app in go using GoFiber and a postgres
database with Go database connection utilities

Utilitises: 
- godotenv
- lib/pq
- gofiber
- database/sql

## To Run

- Install a postgres instance
- Define credentials in .env
